# クイックスタート

この章では、`ccs` をインストールし、数分で Claude Code API エンドポイントの管理を始める方法を説明します。

## クイックナビゲーション

1. **[インストールガイド](installation.md)を読む** —— システム要件とインストール方法。
2. **2 つの機能を理解する** —— [機能](../features/)でエンドポイント切り替えとアップデート確認を確認。
3. **コマンドを習得する** —— [CLI コマンド](../cli/)で各コマンドとオプションを確認。

## 推奨ワークフロー

### 初回利用

1. `npx @xwm111/ccs` を実行（または `npm i -g @xwm111/ccs` でグローバルインストール）。
2. `ccs` を実行してインタラクティブメニューを開く。
3. Claude Code API エンドポイント（base URL + 認証方式 + キー）を追加し、有効化する。

### 日常の利用

- `ccs config-switch`（または `ccs cs`）で有効なエンドポイントを切り替え。
- `ccs cs --list` で保存済みのすべての設定を表示。
- `ccs check-updates`（または `ccs check`）で Claude Code と ccs を更新。

> ヒント：引数なしで `ccs` を実行すると、いつでもインタラクティブメニューが開きます。
