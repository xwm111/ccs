# インストールガイド

このガイドでは、`ccs`（Claude Code Switch）のインストールと実行方法を説明します。

## 環境要件

| 要件 | 最低バージョン | 説明 |
|------|----------------|------|
| **Node.js** | 18.x 以上 | CLI の実行に必要 |
| **npm** | Node.js に同梱 | `npx` サポートに必要 |
| **OS** | - | macOS、Linux、Windows（PowerShell/WSL） |

環境を確認：

```bash
node --version
npm --version
npx --version
```

## インストール

### 方法 1：グローバルインストール（推奨）

```bash
npm i -g @xwm111/ccs
```

以降はどこからでも `ccs` を実行できます。

### 方法 2：npx で実行（インストール不要）

```bash
npx @xwm111/ccs
```

## 初回実行

```bash
ccs
```

初回実行時、ccs は使用する表示言語を尋ねます：

```
? 表示言語を選択：
  ❯ 简体中文
    English
```

言語を選択するとインタラクティブメニューが開きます。メニューから Claude Code API エンドポイントの追加、有効なエンドポイントの切り替え、アップデート確認、言語切り替え、アンインストールが行えます。

## インストールの確認

```bash
# ヘルプを表示
ccs --help

# バージョンを表示
ccs --version
```

## 設定の保存場所

ccs は設定を `~/.ccs` に保存します。
